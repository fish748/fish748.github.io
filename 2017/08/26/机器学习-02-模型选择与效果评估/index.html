机器学习 (02) - 模型选择与效果评估

拟合与欠拟合

  • 训练误差 (training error)：模型在训练样本上得到的误差，也叫经验误差 (empirical error)。

  • 泛化误差 (generation)：模型在新样本上的误差。

    我们希望得到的是泛化误差尽可能小的模型。然而，由于事先无法知道新样本是什么样，所以只能去尽量减小训练误差。在训练模型的过程中，实际上我们希望尽可能学习到所有数据的「普遍规律」。但如果过分追求训练误差的最小化，那么有可能模型会把训练样本自身的规模当做了所有数据的共同规律。这样一来，模型在训练样本上表现非常好，但是在新样本上的表现则很差，即训练误差很小，但泛化误差很差，这种情况叫做过拟合。与过拟合相对的叫欠拟合，即模型连「普遍规律」都没有找全。

  • 过拟合：模型的学习能力太过强大，把训练样本本身包含的不太一般的特性都学到了

  • 欠拟合：模型学习的能力太差，连数据普遍的规律都没有学习到

    解决过拟合的方法主要是正则化和交叉验证。

评估方法

通常，为了验证模型的泛化误差，只会把数据中的一部分作为训练数据，而另一部分作为测试数据 (testing set)，在训练数据上训练模型，并在测试数据上验证模型的泛化误差。训练集与测试集的划分方法包括：

  1. 留出法 (hold-out)：用分层采样的方式，从数据集中直接划分训练集与测试集。一般会随机划分多次、重复多次评估后取平均值。
  2. 交叉验证法 (cross validation)：将数据等分成 K 份，每次用其中的 K-1 份作为训练集，用剩下的一份作为测试集，一共训练 K 次，取均值。
  3. 留一法 (leave-one-out)：就是 K = 数据量 M 的交叉验证法。
  4. 自助法 (bootstrapping)：用放回式采样来选取训练集和数据集。

损失函数

模型预测的输出与真实值之间可能相同，也可能不同，损失函数 (lost function) 或代价函数 (cost function) 用来度量预测错误的程度，记做 $ L(Y, f(X)) $，常用的损失函数有：

  1. 0-1 损失函数 (0-1 loss function)

    ​ $L(Y, f(X)) = \begin{cases} 1, &Y \ne f(X)\, 0, &Y = f(X)\end{cases}$

  2. 平方损失函数 (quadratic loss function)

    ​ $L(Y, f(X)) = (Y - f(X))^2$

  3. 绝对损失函数 (absolute loss function)

    ​ $L(Y, f(X)) = |Y - f(X)|$

  4. 对数损失函数 (logarithmic loss function) 或对数似然损失函数 (log-likelihood loss function)

    ​ $L(Y, P(Y|X)) = -\log P(Y|X)$

三种风险

  1. 期望风险 (expected risk)：

    ​ $R_{exp}(f) = Ep[L(Y, f(X))]=\int{-\mathcal{X}\times\mathcal{Y}}L(Y, f(X))P(x, y) dx dy$

    损失函数的期望是理论上模型 $f(x)$ 关于联合分布 $P(X, Y)$ 的平均意义下的损失，成为风险函数 (risk function) 或期望损失 (expected loss)。学习的目的就是选择期望风险最小的模型。

  2. 经验风险 (empirical risk)：

    ​ $$R{emp}(f) = \sum{i=1}^NL(y_i, f(x_i))$$$

    经验风险是模型关于训练样本集的平均损失。根据大数定理，当样本容量 N 趋于无穷大时，经验风险趋于期望风险。当模型是条件概率分布，损失函数是对数损失函数时，经验风险最小化等价于期望风险最小化。比如极大似然估计。

  3. 结构风险 (structural risk)

    ​ $$R{srm}(f) = \sum{i=1}^NL(y_i, f(x_i)) + \lambda J(f)$$

    经验风险最小化在样本容量过小时学习效果不好，容易出现过拟合，结构风险在经验风险上加上表示模型复杂度的正则化项 (regularizer) 或惩罚项 (penalty term)。当模型是条件概率分布，损失函数是对数函数时，结构风险最小化等价于最大后验概率估计。

性能度量

  1. 均方误差 (mean square error)：

    ​ $$E(f; D) =\frac1m\sum_{i=1}^m(f(\boldsymbol{x_i})-y_i)^2$$

    对于数据分布 $\mathcal{D}$ 和概率密度函数 $p(x)$，均方误差可以描述为

    ​ $$E(f; \mathcal{D}) = \int_{\boldsymbol{x}\sim\mathcal{D}}(f(\boldsymbol{x}) - y)^2p(\boldsymbol{x})d\boldsymbol{x}$$

  2. 错误率和精度

错误率是分类错误的样本数占总样本数的比例，精度则是分类正确的样本数占总样本数的比例

错误率：

​ $$E(f; D) =\frac1m\sum_{i=1}^m\mathbb{I}(f(\boldsymbol{x_i}) \ne yi)$$
​ $$E(f; \mathcal{D}) = \int{\boldsymbol{x}\sim\mathcal{D}}\mathbb{I}(f(\boldsymbol{x_i}) \ne y_i)p(\boldsymbol{x})d\boldsymbol{x}$$

精度

​ $$acc(f; D) = 1 - E(f; D)$$
​ $$acc(f; \mathcal{D}) = 1 - E(f; \mathcal{D})$$

  1. 准确率与召回率

    T (True) 和 F (False) 表示预测是否正确

    P (Positive) 和 N (Negative) 表示真实结果是正例还是反例

    • TP (真正例)：真实结果是正例，并且预测出来也是正例，预测正确
    • FN (假反例)：真实结果是正例，但预测结果是反例，预测错误
    • TN (真反例)：真实结果是反例，并且预测出来也是反例，预测正确
    • FP (假真例)：真实结果是反例，但预测结果是正例，预测错误

    准确率 (presesion)、查准率：所有预测结果是正例的数据里面，真正的正例有多少

    ​ $$P = \frac{TP}{TP + FP}$$

    召回率 (recall)、查全率：所有正例里面，被预测正确的有多少

    ​ $$R = \frac{TP}{TP + FN}$$

    ​

F1 度量：同时考虑准确率与召回率

​ $$F1 = \frac{2 \times P \times R}{P + R}$$
